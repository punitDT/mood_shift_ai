# Assets Directory

## Required Assets

### Images
Place the following images in `assets/images/`:
- `splash_logo.png` - App logo for splash screen (512x512 px recommended)

### Animations (Optional)
Place Lottie animation files in `assets/animations/`:
- You can download free Lottie animations from https://lottiefiles.com/

### Fonts
Place Poppins font files in `assets/fonts/`:
- `Poppins-Regular.ttf`
- `Poppins-Medium.ttf`
- `Poppins-SemiBold.ttf`
- `Poppins-Bold.ttf`

You can download Poppins font from Google Fonts: https://fonts.google.com/specimen/Poppins

## Note
These assets are required for the app to build successfully. Make sure to add them before running the app.
